# Uncommented
Find all those pesky public APIs missing documentation.
The intent is to run this on header files.

## Supported checks
### C
- [x] C function declarations
- [x] C inline functions definitions
- [x] C structs with function pointer members

### C++
- [x] C++ member functions (Methods)
    - [x] public member declarations
    - [x] protected member declarations
- [x] C++ class/struct/union declarations (including templated forms)

### Preprocessor
- [x] Preprocessor macros
